PROYECTOS

Log In PHP

Es un log in seguro, implementando la conexión a la base de datos, sanitizando los datos y protegiendo las contraseñas mediante el hashing and salt.


Island Bot para AlbionOnline

Programa hecho en c++ que recolecta los recursos de las islas del juego con este he logrado pagar el premium anual($99USD en precio del juego) en 47hrs de ejecución del programa. Tiene un algoritmo propio para el desplazamiento.


Talk to LSM

Aplicación móvil que traduce mediante un modelo 3D lo que escribes a LSM(lenguaje de señas Mexicana). La interfaz esta desarrollada con Unity, el modelo hecho en blender. También, tiene implementado KNN para el análisis de la información. Así podemos saber en que ámbitos se usan más las oraciones y palabras. Además, tiene un algoritmo de clustering para clasificar las palabras consultadas que no están registradas en la base de datos con la finalidad de asignar prioridad a aquellas palabras que más son consultadas y de esta manera sean las primeras en ser animadas.


Crypto Tranding Tools

Hecho con NodeJS se conecta a la API de Bitso para obtener los precios, analisarlos y tener señales de compra o de venta, si tienes vinculada tu cuenta puedes hacerle caso a la señal y el programa crea la orden directamente.


Contador de visitas

Contador de visitas hecho en PHP. Ofrece varios informes de alto valor agregado, como la cantidad de visitantes unicos, cuantas veces a entrado cada visitante, su tiempo de conexion. y algunas graficas para ver la informacion mas interactiva.
